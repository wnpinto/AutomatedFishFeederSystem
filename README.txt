*Software Engineering I project*

This project is a continuation of the DBMS aquarium project.


=====PROJECT DESCRIPTION=====
The next part was to design a project plan which included the requirements and specifications for an improvement of the system. In our case, we wanted to improve the fish feeder system. The designs included Software Functions, the system constrains, the architectural design and the system module design.
